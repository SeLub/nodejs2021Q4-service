Module BoardRouter

This is the BOARD router to handle server requests. Depending on method (GET, POST, PUT, DELETE) and endpoint (/boards/:boardId) it retrives data from Service Layer and send response to user.

  • Request endpoints: /boards/:boardId
  • Request methods: GET, POST, PUT, DELETE requests
  • Part of the Server Layer
  • Get data from: BoardService
  • Send Data to: MainRouter
